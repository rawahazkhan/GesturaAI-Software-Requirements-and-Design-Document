# GesturaAI: Sign Language Translation & Learning System

## Project Overview
This repository contains the complete **Software Requirements Specification (SRS)** and **Software Design Document (SDD)** for GesturaAI, an accessibility-focused platform designed to bridge communication gaps for the deaf and hard-of-hearing community. 

The project focuses on the theoretical and technical architecture required to translate hand gestures into text/speech in real-time, while providing an interactive learning environment.

## Engineering Artifacts
This documentation covers the full **Software Development Life Cycle (SDLC)**:

**Feasibility Analysis:** Comprehensive evaluation of Technical, Operational, Economic (Estimated Rs. 6,000), Schedule, and Resource feasibility.
**Project Scheduling:** A structured 11-week development timeline covering dataset handling to final refinements.
**UML Modeling:**
    **Use Case Diagrams:** Defining interactions for All Users, Learners, and System-level actors.
    **Activity Diagrams:** Mapping the logic of the gesture recognition and translation pipeline.
    **Class Diagrams:** Architecting the MERN-based system including User, Recognition, and Learning modules.

## System Features (Conceptualized)
* **Sign Language Recognition:** Real-time detection of hand shapes, movement patterns, and facial expressions (NMF).
* **Bidirectional Translation:** A 3D Animated Avatar that performs signs based on text or voice input.
* **Educational Module:** Step-by-step tutorials, fingerspelling lessons, and AI-driven feedback for learners.
* **Responsible AI:** Built-in cultural sign variation support and data privacy protections.
